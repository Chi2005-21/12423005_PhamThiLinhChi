# Thông tin bộ dữ liệu

Bộ dữ liệu được sử dụng trong đồ án này được lấy từ Kaggle:

**Credit Risk Dataset**  
https://www.kaggle.com/datasets/laotse/credit-risk-dataset

Do dung lượng và điều khoản sử dụng của Kaggle, **toàn bộ dữ liệu không được đưa lên GitHub**.

---

## Mô tả dữ liệu

Bộ dữ liệu chứa thông tin về khách hàng vay vốn và trạng thái khoản vay, dùng để xây dựng mô hình dự đoán rủi ro tín dụng.

- Biến mục tiêu: `loan_status`
  - 0: Rủi ro thấp
  - 1: Rủi ro cao

---

## Hướng dẫn tải và sử dụng dữ liệu

### Bước 1: Đăng nhập Kaggle
- Truy cập: https://www.kaggle.com
- Đăng nhập hoặc tạo tài khoản Kaggle

### Bước 2: Tải bộ dữ liệu
- Truy cập link dataset:  
  https://www.kaggle.com/datasets/laotse/credit-risk-dataset
- Nhấn **Download**
- Giải nén file vừa tải về

### Bước 3: Đặt dữ liệu vào thư mục dự án
- Sao chép file **`credit_risk_dataset.csv`**
- Đặt file vào đúng thư mục sau trong project:
data/credit_risk_dataset.csv
